=============
    API
=============

    FastAPI:
        > app = fastapi.FastAPI() ; Inicializamos la FastAPI
        > @app.get("/") ; Asignar una función como API
        > raise fastapi.HTTPException(status_code=404, detail="Mensaje") ; Gestión de errores
        > return ; Lo que devuelve la API
    
    Uvicorn:
        > uvicorn.run(app, host=HOST, port=PORT) ; Abrir el puerto 
    
    Requests:
        > url_inicial = f"{URL_BASE}{ENDPOINT}/(etc)" ; URL + ENDPOINT
        > headers = {"api_key": API_KEY} ; Si hay 
        > respuesta = requests.get(url_inicial, headers=headers) ; Petición
        > respuesta.json() ; Para que no de error, debe de convertirse en JSON
